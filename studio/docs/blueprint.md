# **App Name**: MediaFlow

## Core Features:

- User Profiles: Users can create and manage their profiles.
- Content Upload and Sharing: Users can upload and share various media types (video, photos).
- Content Sorting: Sort content based on 'newest', 'popular', and 'most viewed' criteria.
- Customizable Feeds: Users can customize their content feeds based on followed users.
- Commenting System: Users can comment on media content.

## Style Guidelines:

- Primary color: Desaturated blue (#5A7D9A) to complement the dark scheme.
- Background color: Dark gray (#282828) to set a modern and sleek tone.
- Accent color: Lighter, brighter blue (#7B90A6) for interactive elements and highlights, ensuring visibility against the dark background.
- Body text: 'Inter' sans-serif font for a clean, modern, and readable interface.
- Headlines: 'Space Grotesk' sans-serif font for a clean, modern, and readable interface.
- Simple, outlined icons in the accent color for key actions.
- Grid-based layout for media display, ensuring a consistent and visually appealing presentation.
